<canvas-text>

A lightweight web component that rasterizes layered rich text and images onto a canvas. Perfect for memes, OG cards, and sticker tools.

View Demos API Documentation

Features

  • Layered Compositing

    Z-ordered image and text slots. Stack backgrounds and text layers in any order with the simple slot grammar.

  • Meme-Ready

    Slot grammar designed for top/bottom text over a background image. slot="background", slot="text-1", slot="text-2".

  • Theme Bridge

    theme="inherit" picks up surrounding CSS tokens — color, font-family, font-size — so the canvas matches your design system automatically.

  • Export to Image

    toBlob() and toDataURL() produce PNG, JPEG, or WebP. Download a rendered meme or upload an OG card in a single call.

  • Reactive

    Slot and attribute mutations auto-trigger a coalesced re-render via MutationObserver and requestAnimationFrame.

  • Accessible

    Slotted HTML stays in the light DOM, visually hidden so screen readers consume it as normal text. Canvas is aria-hidden.

Quick Start

Installation

Install via npm:

npm install @profpowell/canvas-text render-tag

render-tag is a peer dependency — add it alongside the component.

Basic Usage

<script type="importmap"> { "imports": { "render-tag": "https://esm.sh/render-tag@0.1.7" } } </script> <script type="module" src="https://unpkg.com/@profpowell/canvas-text"></script> <canvas-text width="600" compose="text-only"> <strong>Hello</strong> world </canvas-text>

Layered (Meme) Usage

<canvas-text width="500" height="500"> <img slot="background" src="doge.jpg" crossorigin="anonymous"> <div slot="text-1" style="text-align:center; font-size:48px; font-weight:bold; color:white; -webkit-text-stroke:2px black; padding-top:20px;"> TOP TEXT </div> <div slot="text-2" style="text-align:center; font-size:48px; font-weight:bold; color:white; -webkit-text-stroke:2px black; padding-top:400px;"> BOTTOM TEXT </div> </canvas-text>
