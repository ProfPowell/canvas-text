Basic Text Rendering

Use compose="text-only" to render a single rich-text layer from the element's default slot. No layer pipeline overhead.

Live Demo

Hello, world!

Source

<canvas-text width="500" height="120" compose="text-only">
  <p style="padding: 20px 24px; font-size: 28px; margin: 0;">
    <strong>Hello</strong>, <em>world</em>!
  </p>
</canvas-text>

Layered Composition

Combine slot="background" with slot="text-1" and slot="text-2" to composite text over an image — classic meme layout.

Live Demo

TOP TEXT
BOTTOM TEXT

Source

<canvas-text width="500" height="500" theme="none">
  <img slot="background" src="./test/fixtures/red-square.png"
       crossorigin="anonymous" alt="Red background">
  <div slot="text-1"
       style="text-align:center; font-size:52px; font-weight:bold;
              color:white; -webkit-text-stroke:2px black;
              padding-top:24px;">
    TOP TEXT
  </div>
  <div slot="text-2"
       style="text-align:center; font-size:52px; font-weight:bold;
              color:white; -webkit-text-stroke:2px black;
              padding-top:400px;">
    BOTTOM TEXT
  </div>
</canvas-text>

Use padding-top for vertical placement of bottom text — position:absolute is not supported by render-tag. See CSS support for the full list.

Why theme="none"? When you fully style your text layers (color, font-size, stroke) the default theme="inherit" can introduce a stroke/fill misalignment artifact. For tightly-styled meme text, theme="none" hands render-tag the raw markup with no inherited wrapper.

Background Z-Stacking

slot="background-N" stacks image layers by integer N. Higher N paints on top. Text at the same N renders above its background peer.

Live Demo

z=3 text

Source

<canvas-text width="400" height="400" theme="none">
  <!-- z=1: full red background -->
  <img slot="background-1" src="./test/fixtures/red-square.png"
       crossorigin="anonymous" alt="Red background at z=1">

  <!-- z=2: blue square overlaps red -->
  <img slot="background-2" src="./test/fixtures/blue-square.png"
       crossorigin="anonymous" alt="Blue square at z=2"
       style="width:200px; height:200px;">

  <!-- z=3 text: renders above both backgrounds -->
  <div slot="text-3"
       style="text-align:center; font-size:32px; font-weight:bold;
              color:white; -webkit-text-stroke:1px black;
              padding-top:160px;">
    z=3 text
  </div>
</canvas-text>

Theme Inherit

theme="inherit" (default) copies CSS tokens from the host element into each text layer. theme="none" uses render-tag's defaults only.

Live Demo

theme="inherit" — picks up surrounding color & font

Theme: inherit

theme="none" — render-tag defaults

Theme: none

Source

<!-- Wrap canvas-text in a styled ancestor -->
<div style="color: steelblue; font-size: 28px;
            font-weight: 600; font-family: Georgia, serif;">

  <!-- theme="inherit": canvas picks up steelblue + Georgia -->
  <canvas-text width="320" height="100"
               theme="inherit" compose="text-only">
    <div style="padding: 16px 20px;">Theme: inherit</div>
  </canvas-text>

  <!-- theme="none": render-tag defaults regardless of host style -->
  <canvas-text width="320" height="100"
               theme="none" compose="text-only">
    <div style="padding: 16px 20px;">Theme: none</div>
  </canvas-text>
</div>

Allowlisted properties: color, font-family, font-size, font-weight, font-style, line-height, letter-spacing, text-align, background-color. See Theme Bridge API.

Export to Image

toBlob() returns a Promise<Blob>; toDataURL() returns a base64 string synchronously. Both reflect the last completed render.

Live Demo

Export me as PNG or JPEG!

Source

<canvas-text id="export-demo" width="500" height="160" compose="text-only">
  <div style="padding: 32px; font-size: 32px; font-weight: bold;
              text-align: center; color: #0969da;">
    Export me as PNG or JPEG!
  </div>
</canvas-text>

<button id="btn-png">Download PNG</button>
<button id="btn-jpeg">Download JPEG</button>

<script type="module">
  const el = document.getElementById('export-demo');

  async function download(type, ext) {
    const blob = await el.toBlob(type);
    const url = URL.createObjectURL(blob);
    const a = Object.assign(document.createElement('a'), {
      href: url, download: `canvas-text-export.${ext}`
    });
    a.click();
    URL.revokeObjectURL(url);
  }

  document.getElementById('btn-png').addEventListener('click',
    () => download('image/png', 'png'));
  document.getElementById('btn-jpeg').addEventListener('click',
    () => download('image/jpeg', 'jpg'));
</script>

Accessibility

The slotted HTML is visually hidden but AT-readable. The canvas is aria-hidden="true". Set alt for a concise label.

Live Demo

Hello from canvas-text!

How it works

When alt is set the canvas receives aria-label="A friendly hello message" and the slotted fallback is suppressed so AT reads the alt once, not both. Without alt, the slotted children are the AT label — the canvas is aria-hidden and the visually-hidden light DOM is consumed directly.

Visually-hidden DOM snippet (inspect to verify) 
<canvas-text alt="A friendly hello message">
  <!-- slotted children stay in light DOM, hidden via clip-path -->
  <p style="...">Hello from <strong>canvas-text</strong>!</p>

  <!-- canvas is aria-hidden, gets aria-label from alt -->
  <canvas aria-hidden="true" aria-label="A friendly hello message"></canvas>
</canvas-text>

  </p>
</canvas-text>

Reactive Updates

Slot and attribute mutations trigger an automatic coalesced re-render. Type in the box or swap the background — the canvas updates without any manual render() call.

Live Demo

Hello, canvas-text!

Source

<input id="dynamic-text" type="text" value="Hello, canvas-text!">
<select id="dynamic-bg">
  <option value="red">Red square</option>
  <option value="blue">Blue square</option>
  <option value="none">None</option>
</select>

<canvas-text id="dynamic-demo" width="500" height="200" theme="none">
  <img id="dynamic-bg-img" slot="background"
       src="./test/fixtures/red-square.png" crossorigin="anonymous" alt="">
  <div id="dynamic-text-layer" slot="text-1"
       style="padding: 60px 32px; font-size: 36px; font-weight: bold;
              text-align: center; color: white; -webkit-text-stroke: 1px black;">
    Hello, canvas-text!
  </div>
</canvas-text>

<script type="module">
  const textInput = document.getElementById('dynamic-text');
  const bgSelect  = document.getElementById('dynamic-bg');
  const bgImg     = document.getElementById('dynamic-bg-img');
  const textLayer = document.getElementById('dynamic-text-layer');

  textInput.addEventListener('input', () => {
    textLayer.textContent = textInput.value;
  });

  bgSelect.addEventListener('change', () => {
    const fixtures = './test/fixtures/';
    if (bgSelect.value === 'none') {
      bgImg.removeAttribute('src');
    } else {
      bgImg.src = fixtures + bgSelect.value + '-square.png';
    }
  });
</script>
